# N-body gravitational simulator
3D N-body simulator designed to efficiently process large numbers of objects

## Features
- Octree spatial partitioning
- Barnes-Hut algorithm for reducing computational complexity
- Leapfrog symplectic integrator for minimizing fluctuations in system energy between simulation steps
- Unit system using Earth–Moon distances, Earth masses, and Earth days
- Collision handling models

## Architecture
This project follows a modular monolith architecture.

## Notes
All aspects of this project are subject to change as I learn more about software development and numerical simulations.  
Some implementations may be experimental, suboptimal, or refactored later as the project evolves, including parts of the architecture.
